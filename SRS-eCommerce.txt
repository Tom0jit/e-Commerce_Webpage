Requirements Elicitation and Analysis
-------------------------------------
SRS (Software Requirements Specification Document)
----------------------------------------
1. The Login pagewill ask for a USERNAME and PASSWORD combination which will be verified before 
the user is allowed to enter the website.
2. The Sign Up form will contain EMAIL, USERNAME and PASSWORD fields, all of which are compulsory
to enter.
3. The user should be able to view the products as a list. It should display the product PICTURE,
NAME, CATEGORY, and a SMALL DESCRIPTION. The user should be able to CLICK on a product to view
full details.
4. This page displays ALL THE DETAILS OF A PRODUCT.
Details : PICTURE, NAME, CATEGORY, SUBCATEGORY, PRICE, UNITS, DESCRIPTION.
5. The user should be allowed to ADD a product to the CART from the Details Page.
6. The user can VIEW all the products that they have added along with the total price of the 
entire order.
7. The order will be processed after the user makes payment successfully.
